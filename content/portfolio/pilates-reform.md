**The brief.** A reformer pilates studio wanted a website that felt as composed and premium as a session on the apparatus itself. The old site buried the essentials, so prospective members couldn't quickly see what the classes were, how the method worked, or what membership cost.

**The approach.** We built a light, minimal layout in chalk, black and clay tones that lets the studio's own photography breathe. Hanken Grotesk keeps the interface calm and legible, while Fraunces adds the chic editorial italics that carry the brand's premium feel. Every section answers one question — method, classes, membership — with no clutter in between.

**Why it works.** The restraint reads as confidence: nothing competes for attention, so the few clear calls to action land. Distinct sections for the reformer principle, the class formats and transparent pricing mean a first-time visitor can decide to book in a single, unhurried scroll.
